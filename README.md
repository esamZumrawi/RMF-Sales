# RMF-Sales Analysis Project
RFM Sales Analysis project using SQL Server: consolidates monthly sales, calculates Recency, Frequency, Monetary metrics, ranks and scores customers, and creates BI-ready segments like Champions, Loyal VIPs, and At-Risk, enabling data-driven insights for marketing, retention, and sales strategy.

# Overview
This project performs a comprehensive RFM (Recency, Frequency, Monetary) analysis on sales data to evaluate customer purchasing behavior and segment customers for actionable insights. It consolidates monthly sales tables, calculates RFM metrics, assigns ranks and decile scores, and generates total RFM scores to identify high-value, loyal, and at-risk customers. The final output is a BI-ready customer segmentation table suitable for visualization in tools like Power BI or Tableau.

# Features
* Data Consolidation: Combines multiple monthly sales tables into a single dataset.
* RFM Metrics: Computes Recency, Frequency, and Monetary values for each customer.
* Ranking & Deciles: Assigns ranks and decile scores to measure customer value.
* Total RFM Score: Aggregates individual RFM scores for overall customer evaluation.
* Customer Segmentation: Classifies customers into categories: Champions, Loyal VIPs, Potential Loyalists, Promising, Engaged, Requires Attention, At-Risk, and Lost/Inactive.
* BI-Ready: Creates a clean, structured output for dashboards and reporting.

# Technologies
* SQL Server – Data aggregation, ranking, and segmentation.
* RFM Framework – Customer scoring and segmentation methodology.
* Business Intelligence – Supports reporting and dashboard integration.

# Use Cases
* Identify top-value customers for targeted marketing.
* Detect at-risk customers for retention campaigns.
* Support data-driven decision-making in sales and marketing strategies.
